# Cryptography Algorithms Implementation using Python

## Overview

This project demonstrates the implementation of fundamental cryptographic algorithms using Python. The objective is to understand encryption, decryption, and hashing techniques commonly used in cybersecurity for secure communication and data protection.

## Objectives

* Understand basic cryptography concepts
* Implement encryption and decryption algorithms
* Generate secure hash values using SHA256
* Learn practical applications of cryptography in cybersecurity

## Tools Used

* Python 3
* PyCryptodome
* Kali Linux

## Algorithms Implemented

### 1. Caesar Cipher

The Caesar Cipher is a classical encryption algorithm that shifts characters in a message by a specified number of positions.

**Features:**

* Text Encryption
* Text Decryption
* User-defined shift value

### 2. SHA256 Hash Generator

SHA256 is a cryptographic hashing algorithm that generates a fixed-length hash value from input data.

**Features:**

* Secure hash generation
* Data integrity verification
* One-way hashing mechanism

### 3. AES Encryption and Decryption

Advanced Encryption Standard (AES) is a widely used symmetric encryption algorithm for protecting sensitive data.

**Features:**

* AES Encryption
* AES Decryption
* Secure symmetric key usage
